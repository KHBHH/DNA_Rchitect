createCytoscapeJsNetwork {rcytoscapejs}R Documentation

Generate a CytoscapeJS compatible network

Description

Generate a CytoscapeJS compatible network

Usage

createCytoscapeJsNetwork(nodeData, edgeData, nodeColor = "#888888",
  nodeShape = "ellipse", nodeHeight = "70", nodeWidth = "70",
  nodeLabelColor = "#FFFFFF", edgeColor = "#888888",
  edgeSourceShape = "none", edgeTargetShape = "triangle", nodeHref = "")

Arguments

nodeData

a data.frame with at least two columns: id and name; Any additional columns from nodeData will be passed to CytoscapeJS. If 'x' and 'y' columns are specified and the layout in rcytoscapejs function is set to 'preset' then the user-specified locations will be used.

edgeData

a data.frame with at least two columns: source and target

nodeColor

a hex color for nodes (default: #666666)

nodeShape

a shape for nodes (default: ellipse)

nodeHeight

node height (default: 30)

nodeWidth

node width (default: 30)

nodeLabelColor

node label color (default: #FFFFFF)

edgeColor

a hex color for edges (default: #666666)

edgeSourceShape

a shape for arrow sources (default: none)

edgeTargetShape

a shape for arrow targets (default: triangle)

nodeHref

a link associated with the node (appears in a tooltip)

Value

a list with two entries:

  • nodes a JSON string with node information compatible with CytoscapeJS

  • edges a JSON string with edge information compatible with CytoscapeJS

If required columns (see parameters) are not provided, an error will be thrown

See http://cytoscape.github.io/cytoscape.js/ for shape details

See Also

rcytoscapejs

Examples

id <- c("Jerry", "Elaine", "Kramer", "George")
name <- id
nodeData <- data.frame(id, name, stringsAsFactors=FALSE)

source <- c("Jerry", "Jerry", "Jerry", "Elaine", "Elaine", "Kramer", "Kramer", "Kramer", "George")
target <- c("Elaine", "Kramer", "George", "Jerry", "Kramer", "Jerry", "Elaine", "George", "Jerry")
edgeData <- data.frame(source, target, stringsAsFactors=FALSE)

network <- createCytoscapeJsNetwork(nodeData, edgeData)
